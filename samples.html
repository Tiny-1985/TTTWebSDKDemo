TTTWebSDK samples

如下示例代码，展示了 TTTWebRTC API 的使用，供开发者参考。

注：TTTWebRTC API 详细说明，请参考 TTTWebRTC API

加入房间(join)

web端加入 3T 房间

  • join - 加入3T房间

创建本地流 & 预览

createStream 创建本地流

  • createStream - 创建流

RTC推流

publish - 向 3T RTC 推流

  • publish - 音视频RTC推流
  • publish - 纯音频RTC推流
  • publish - 屏幕流RTC推流
  • publish - 向RTC推流 - 多路视频流

RTC推流-CDN

RTC推流，经由 3T RTC 旁路推 CDN

  • publish - rtmp - 房间内流经 RTC 推CDN
  • publishScreen - rtmp - 屏幕流经 RTC 推CDN

订阅媒体流

从 3T RTN 订阅房间内其他用户已发布的媒体流

  • subscribe - 订阅 RTC 流
